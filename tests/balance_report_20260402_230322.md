## Methodology

- **Equal budget**: both players receive the same coin budget per match.
- **Heuristic placement**: tanks → front row, melee → mid row, ranged → back row.
- **Unit vs unit matrix**: mono-army of each type vs mono-army of every other type, same budget.
- **Random matches**: both sides compose randomly from all affordable units.
- **Archetype matches**: hand-crafted strategic compositions (swarm, wall, tank+ranged, etc.).
- **Stat search**: each unit tested at ±25%/±50% health and ±1/±2 damage vs its own unmodified mirror.
- **Upgrade paths**: all 1- and 2-upgrade combinations tested vs unupgraded same unit.
- **Balance thresholds**: >62% = overtuned, <38% = undertuned.
